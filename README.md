AppAcademia

AppAcademia é uma aplicação web que calcula o IMC do usuário e, com base no resultado, exibe treinos e dietas personalizadas.
Possui sistema de login e cadastro, persistência de dados com localStorage, navegação suave entre seções e layout responsivo para desktop e mobile.
Projeto desenvolvido para praticar HTML, CSS e JavaScript, com foco em manipulação do DOM, lógica condicional e organização de dados.

Tecnologias Utlizadas

HTML5
CSS3 (Flexbox, Media Queries)
JavaScript
localStorage
